## 📋 详细流程

**工具**：`auto_screenshot.py`

| 其他门派 | 各 50 张 | ⭐⭐⭐ |

---

### 第二阶段：自动标注（10 分钟）

**工具**：`auto_labeler.py`

**功能**：
- ✅ 批量预测 NPC 位置
- ✅ 生成 YOLO 格式标注
- ✅ 统计标注信息

**标注格式**（YOLO 标准）：
```
# 每行一个 NPC
class_id x_center y_center width height
0 0.4523 0.6234 0.0523 0.0834
1 0.7234 0.3456 0.0634 0.0923
```

**类别对应**：
- `0` - 门派师傅
- `1` - 门派师兄
- `2` - 门派守卫
- `3` - 杂货店老板
- ...（共 35 类）

---

### 第三阶段：人工修正（30-60 分钟）

**工具**：LabelImg

**安装**：
```bash
pip install labelImg
labelImg
```

**操作步骤**：
1. 打开 LabelImg
2. 加载图片目录：`dataset/npc_images`
3. 加载标注目录：`dataset/yolo_dataset/labels`
4. 逐张检查
5. 修正错误标注

**修正要点**：
- ✅ 删除误检（不是 NPC 的框）
- ✅ 补充漏检（有 NPC 没框）
- ✅ 调整框位置（要框住整个 NPC）
- ✅ 修正类别（选择正确的 NPC 类型）

**快捷键**：
- `A` - 上一张
- `D` - 下一张
- `W` - 绘制框
- `Del` - 删除框
- `Ctrl+S` - 保存

---

### 第四阶段：训练模型（1-2 小时）

**工具**：`train_yolo.py`

**训练配置**：
```python
epochs = 100          # 训练轮数
batch_size = 16       # 批次大小
img_size = 640        # 图像尺寸
model = 'yolov8n.pt'  # 模型（nano 版本）
```

**训练过程**：
```
Epoch   GPU_mem   box_loss   cls_loss   Instances   Size
  1/100    0.52G      1.234      0.567         12      640
  2/100    0.52G      1.123      0.489         15      640
  ...
100/100    0.52G      0.234      0.123         18      640
```

**输出**：
- `best.pt` - 最佳模型
- `last.pt` - 最后一轮模型
- `results.png` - 训练曲线
- `results.csv` - 训练数据

---

### 第五阶段：评估测试（10 分钟）


## 🆘 常见问题

### Q: 我没有 GPU，能训练吗？
**A**: 可以，但速度较慢
- CPU 训练：2-4 小时
- GPU 训练：30-60 分钟
- 建议使用 Google Colab（免费 GPU）

### Q: 标注太麻烦，有捷径吗？
**A**: 
1. 先用模拟标注（自动）
2. 训练初始模型
3. 用模型预标注
4. 只修正错误部分（省时 70%）

### Q: 训练失败怎么办？
**A**: 
1. 检查依赖是否安装
2. 检查数据集格式
3. 查看错误日志
4. 参考文档：`docs/YOLO 训练完整流程.md`
