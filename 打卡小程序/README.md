# 上班点数记录小程序 📅

> 一款简洁美观的微信小程序，用于记录每天上班的"点数"，自动统计收入 💰

## 功能特点

### 核心功能
- 📅 **日历打卡** - 点击日期快速录入点数
- 🌴 **休假标记** - 一键标记休假，不计入统计
- 📊 **统计页面** - 实时查看总点数和金额
- ⚙️ **设置灵活** - 自定义每点单价、快捷选项等

### 高级功能（8个额外功能）
1. 🔥 **连续打卡** - 显示连续打卡天数
2. 🔔 **每日提醒** - 定时提醒录入点数
3. 📊 **批量录入** - 快速补录历史记录
4. 🎯 **月度目标** - 设定目标并跟踪进度
5. 🧮 **薪资计算器** - 快速估算月薪/年薪
6. 📤 **月度报告** - 生成分享海报
7. 📝 **备注模板** - 快速添加常用备注
8. 📚 **多账本** - 分类记录不同工作

## 设计风格
- 🌿 绿色清新风格
- 🎨 渐变卡片设计
- 📱 原生小程序体验

## 技术栈
- WXML + WXSS + JavaScript
- 微信小程序原生框架
- 本地存储（Storage）

## 使用说明

### 安装
1. 下载代码到本地
2. 使用微信开发者工具打开项目
3. 填入自己的 AppID
4. 编译运行

### 配置
- 修改 `app.js` 中的 `pricePerPoint` 设置每点单价（默认 ¥18.6）
- 在 `设置` 页面调整快捷选项和工作日
