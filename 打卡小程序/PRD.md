## 一、产品概述

### 1.1 背景

主人目前用在线文档记录每天上班的"点数"，在手机上查看和录入不方便，因此希望开发一款微信小程序，实现随时随地快速打点、统计收入。

### 1.2 目标用户

个人使用，单人模式，无需注册登录。

### 1.3 核心价值

| 痛点 | 解决方案 |
|------|---------|
| 在线文档手机端操作繁琐 | 原生小程序，一键点击即可录入 |
| 看不到当天还剩多少点 | 日历上直接显示每天点数 |
| 月末手算金额麻烦 | 自动统计页，实时换算金额 |
| 假期不好区分 | 一键标记休假，自动排除计算 |

### 2.2 首页 — 打卡日历

#### 2.2.1 日历视图

- 默认显示**当月**日历，支持左右滑动/翻页切换月份
- 每个日期格显示：
  - 日期数字（大）
  - 今日点数（小字，红色/绿色醒目）
  - 状态角标：
    - 🔴 未录入工作日
    - ✅ 已录入
    - 🌴 休假
    - 📅 未来日期（置灰）
  - 今天用高亮边框标识

#### 2.2.2 录入弹窗（核心交互）

**触发**：点击任意日期格  

弹窗内容：
```
┌─────────────────────────────┐
│  📅  4月23日（周三）         │
│                             │
│  今日点数                   │
│  [  +  ] [ 8.5 ] [  -  ]   │  ← 支持手动输入 + 步进调整
│                             │
│  常用快选：                  │
│  [ 8 ] [ 8.5 ] [ 9 ] [ 半 ] │  ← 可在设置中自定义
│                             │
│  ☑ 标记为休假（不计入统计）  │
│                             │
│  备注（可选）：___________   │
│                             │
│  [  取消  ]  [  保存  ✓  ]  │
└─────────────────────────────┘
```

**逻辑说明**：
- 点数支持小数（如 8.5、7.5）
- 勾选休假后，点数输入框自动禁用
- 已录入的日期再次点击，弹窗显示历史数据可修改
- 点击弹窗外区域 = 取消（不保存）

#### 2.2.3 月度小结条

日历顶部悬停显示当月统计概览：
```
4月 | 已录入 18天 | 合计 152.5点 | 预计 ¥3,050
```

## 四、数据结构设计

### 4.1 本地存储（wx.setStorageSync）

```json
// 存储 key: "records"
{
  "2026-04-23": {
    "points": 8.5,
    "isVacation": false,
    "note": "加班1小时",
    "createdAt": "2026-04-23T18:30:00",
    "updatedAt": "2026-04-23T18:30:00"
  },
  "2026-04-21": {
    "points": 0,
    "isVacation": true,
    "note": "",
    "createdAt": "2026-04-21T09:00:00",
    "updatedAt": "2026-04-21T09:00:00"
  }
}

// 存储 key: "settings"
{
  "pricePerPoint": 20,
  "quickPoints": [8, 8.5, 9, 4],
  "halfDayPoints": 4,
  "workDays": [1, 2, 3, 4, 5],
  "reminderEnabled": true,
  "reminderTime": "18:00",
  "monthlyTarget": 180
}
```

### 4.2 数据操作

- **存储**：`wx.setStorageSync` / `wx.setStorage`
- **读取**：`wx.getStorageSync` / `wx.getStorage`
- **备份/导出**：将 JSON 数据生成文件，通过 `wx.shareFileMessage` 分享
- **数据上限**：微信本地存储上限 10MB，单个日期记录约 200 字节，10年数据约 700KB，完全够用

## 五、UI 设计规范

### 5.1 整体风格

### 5.2 关键交互
- 弹窗使用从底部滑入动画（半屏弹窗）
- 保存成功后日期格点数立即更新（无需刷新）
- 翻月时日历整体左右滑动切换

### 5.3 适配
- 支持 iPhone、Android 主流机型
- 安全区域适配（底部 Home 条）
