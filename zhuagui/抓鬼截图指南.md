## 🎯 地点 5&6：战斗中 - 主鬼 + 小怪（350 张）

### 战斗截图方式

#### 方法 1：自动截图（推荐）
```bash
python tools/auto_screenshot.py
# 选择"战斗截图模式"
# 设置数量：350 张
# 间隔：2 秒（战斗很快）
```

#### 方法 2：手动截图
```
1. 组队抓鬼（自己当队长）
2. 进入战斗后按 F12 截图
3. 每场战斗截图 5-10 张
4. 需要 30-40 场战斗
```

### 战斗截图要点
- ✅ 主鬼（最大的那个）
- ✅ 小怪（周围的鬼）
- ✅ 包含战斗界面
- ✅ 包含血条
- ✅ 包含回合数

## ❓ 常见问题

### Q1: 找不到马副将怎么办？
**A**: 
- 使用 ALT+G 输入坐标 (180, 220)
- 或在长安城寻找（靠近衙门）

### Q2: 怎么去驿站？
**A**: 
- 在长安城找到驿站老板
- 对话选择进入驿站

### Q3: 怎么去地府？
**A**: 
- 长安城驿站 → 大唐国境
- 从大唐国境跑到地府入口
- 进入地府

### Q4: 战斗中截图太暗怎么办？
**A**: 
- 调整游戏亮度
- 或关闭战斗特效

### Q5: 1100 张截图需要多久？
**A**: 
- NPC 截图：38 分钟
- 战斗截图：35 分钟
- 总计：约 1 小时 13 分钟
