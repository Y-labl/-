# 🌊 和鲸社区（Heywhale）训练指南

## 📋 步骤说明

### 1️⃣ 注册账号
访问：https://www.heywhale.com/
- 点击"登录/注册"
- 使用手机号或微信注册

### 2️⃣ 创建项目
1. 登录后点击顶部导航栏的"项目"
2. 点击"创建项目"
3. 选择"Jupyter Notebook"
4. 项目名称：`zhuagui-yolo-training`
5. 点击"创建"

### 3️⃣ 上传数据
1. 在项目页面，点击"上传文件"
2. 上传 `annotation_100.zip`（约 80MB）
3. 上传 `zhuagui_heywhale.ipynb`

### 4️⃣ 配置环境
1. 点击项目右上角的"设置"
2. 选择"计算资源"
3. 选择"GPU"（免费额度足够用）
4. 镜像选择：`Python 3.10` 或 `PyTorch 2.0+`

### 5️⃣ 开始训练
1. 回到项目主页
2. 点击 `zhuagui_heywhale.ipynb`
3. 点击"运行环境" → "启动"
4. 等待环境启动（约 1-2 分钟）
5. 从上到下依次点击每个代码单元格的"运行"按钮

### 6️⃣ 等待训练完成
- GPU 训练时间：5-10 分钟
- 训练完成后会自动显示结果

### 7️⃣ 下载模型
训练完成后：
1. 在左侧文件浏览器找到 `trained_model/zhuagui_v1/weights/best.pt`
2. 右键点击 → "下载"
3. 同样下载 `best.onnx`（如果有）

## 📊 预期结果

- **mAP50**：0.7-0.85
- **模型大小**：约 6MB
