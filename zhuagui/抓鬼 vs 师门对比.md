## 🎮 实战对比（修正后）

### 师门任务流程
```
1. 找师傅领任务
2. 识别任务类型（35 种可能）
   - 如果是寻物：
     → 识别物品类型
     → 找 NPC 购买
     → 或从背包使用
   - 如果是巡逻：
     → 找巡逻点
     → 点击巡逻
   - 如果是捕捉：
     → 找商会买召唤兽
     → 或野外捕捉
   - 如果是示威：
     → 找地点
     → 使用技能
   - 如果是送信：
     → 找 NPC
     → 对话交付
   - 如果是找人：
     → 找 NPC
     → 对话交付
3. 返回找师傅交任务
4. 重复 10 轮

复杂度：⭐⭐⭐⭐⭐
```

### 抓鬼任务流程（修正后）
```
1. 组建队伍（5 人）
2. 找马副将领双倍
3. 飞到驿站
4. 找驿站老板传送到大唐国境
5. 从大唐国境跑到地府
6. 找黑无常领鬼王任务
7. 出来找钟馗领抓鬼任务
8. 使用天眼通符（显示真实坐标）
9. 识别任务栏（地图 + 坐标）
10. 寻路到目标位置
11. 击杀鬼/鬼王
12. 返回驿站 → 大唐国境 → 地府（循环）

复杂度：⭐⭐
```
